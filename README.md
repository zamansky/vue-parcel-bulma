# vue-template


> A sumple Vue 2.0 template.


### Usage

This is a project template for [vue-cli](https://github.com/vuejs/vue-cli).

``` bash
$ npm install -g vue-cli
$ vue init zamansky/vue-template my-project
$ cd my-project
$ npm install 
$ npm run dev
```

### What's Included

- parcel-bundler (instead of webpack)
- vue-router
- vuex
- buefy/bulma
